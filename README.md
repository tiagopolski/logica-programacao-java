# Lógica de Programação em Java

Este repositório contém exercícios e atividades acadêmicas desenvolvidas durante os estudos em programação, com foco em lógica de programação e fundamentos da linguagem Java.

Os códigos foram criados como parte de atividades de faculdade e exercícios práticos para fixação de conceitos básicos da linguagem.

## Conteúdos praticados
- Estruturas condicionais (if/else)
- Operador ternário
- Estruturas de repetição
- Entrada e saída de dados
- Operações matemáticas básicas
- Validação de dados
- Resolução de problemas com base em enunciados

## Tipos de exercícios desenvolvidos
- Verificação de número positivo ou negativo
- Cálculo de soma e diferença entre valores
- Cálculo de salário com base em horas trabalhadas
- Cálculo de renda e impostos
- Cálculo de área e raio
- Verificação de paridade (par ou ímpar)
- Verificação de múltiplos
- Cálculo de valores em lanchonete a partir de código e quantidade
- Verificação de intervalo de números
- Cálculo de duração de jogo
- Identificação de quadrantes no plano cartesiano (eixo X e Y)
- Mensagem de saudação (bom dia, boa tarde, boa noite)
- Resolução de problemas condicionais diversos

## Estrutura do repositório
Os arquivos representam exercícios independentes, cada um abordando um problema específico utilizando conceitos básicos da linguagem Java.

## Objetivo
Praticar lógica de programação e desenvolver raciocínio lógico aplicado à resolução de problemas computacionais utilizando a linguagem Java.
